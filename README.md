# ClassiCube2Minetest

Converts Classicube Blocks (for now) to minetest

### Required 

* Bash
* Python 3.x
* nbtlib (python) for nbt command

```bash
$ pip install "nbtlib==1.12.1"
```

### Features

* Sky Color
* Cloud Color
